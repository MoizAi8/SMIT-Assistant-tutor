# Assignment: Temperature Converter

## Due Date: 2026-06-20

## Description
Write a Python program that converts temperatures between Celsius and Fahrenheit.

## Requirements
1. Define a function `celsius_to_fahrenheit(celsius)` that converts Celsius to Fahrenheit
2. Define a function `fahrenheit_to_celsius(fahrenheit)` that converts Fahrenheit to Celsius
3. The program should prompt the user to enter a temperature value
4. The program should ask whether the input is Celsius or Fahrenheit
5. Display the converted result clearly
6. Use proper function names and variable names
7. Include a main() function that runs the program
8. Add comments to explain your code
9. Handle invalid input gracefully (e.g., non-numeric values)

## Formula
- Celsius to Fahrenheit: F = (C * 9/5) + 32
- Fahrenheit to Celsius: C = (F - 32) * 5/9

## Example Output
```
Enter temperature: 100
Is this Celsius or Fahrenheit? (C/F): C
100°C = 212.0°F
```
