# SMIT-Assistant-tutor-
This project is a teaching assistant agent designed to help beginner programming students understand and evaluate their code. The agent reviews code submissions based on given instructions and rubric, provides clear and beginner-friendly feedback, and suggests improvements to help students learn effectively.
It is built to simulate an intelligent tutor that guides learners step-by-step, making programming concepts easier to understand. The system focuses on fairness in grading, clarity in explanations, and educational support.
🚀 Features:
Code review based on assignment rubric
Beginner-friendly feedback
Educational explanations
Structured evaluation system
Extensible agent architecture.
